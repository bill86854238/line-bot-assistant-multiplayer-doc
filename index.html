常見問題

Q1：如何設定與刪除提醒？

🎨 新版引導式設定：

現在提供更友善的卡片式引導介面，只需跟著步驟操作即可完成設定，無需記憶複雜格式！

📝 傳統格式設定：

仍可使用傳統格式輸入，例如：

  • 設定1_台北_17:00_1,3,5
  • 設定2_台中_08:00_每天
  • 設定3_高雄_09:00_週末
  • 設定4_新竹_09:00_平日

星期數字對應：0=週日, 1=週一, 2=週二, 3=週三, 4=週四, 5=週五, 6=週六

刪除提醒請輸入「刪除+設定編號」，例如「刪除1」。

Q2：如何查詢目前狀態？

輸入「查詢」即可查看會員等級與使用組別。

或直接輸入車站名稱查詢該站誤點資訊。

Q3：付費方案有什麼功能？

我們提供基礎免費版本和付費進階版本：

  • 免費版本：基本誤點通知、車站查詢功能
  • 付費版本：智能預提醒、歷史統計分析、優先更新頻率、進階介面體驗等專業功能

付費功能旨在提供更完整的通勤助手體驗，免費用戶仍可正常使用基本功能。

Q4：如何升級到付費版本？

可透過以下方式升級：

  • 前往我們的 Gumroad 商店購買會員方案
  • 購買後會收到啟用碼，在 LINE Bot 中輸入即可升級
  • 支援月卡（31天）、季卡（93天）
Q5：付費版本的預提醒多久前會通知？

根據會員等級提供不同的預提醒門檻：

  • 標準會員：誤點超過 3 分鐘才預提醒，可設定 2 個預提醒
  • 進階會員：誤點超過 2 分鐘才預提醒，可設定 5 個預提醒
  • 免費會員：無預提醒功能
Q6：歷史統計資料可以看多久以前的？

歷史統計功能為付費會員專屬，可查看火車延誤的歷史數據分析，具體保存期限依資料庫容量而定，一般可查看數個月的歷史記錄。

Q7：個人化車次記憶是什麼？

系統會記錄您常用的車次和路線，在設定頁面優先顯示您經常搭乘的班次，讓重複設定更加便利。根據您的使用歷史，最多顯示 3 個常用車次供快速選擇。

Q8：到站提醒如何設定？

到站提醒設定方式：

  1. 直接傳送訊息「我要搭 1152 車次從台北到台中」
  2. 或使用選單功能選擇起終點站和車次
  3. 系統會自動追蹤該車次並在到站前提醒您
  4. 提醒會在列車進站時主動推送
Q9：為什麼我沒收到預提醒？

可能的原因：

  • 您是免費會員（無預提醒功能）
  • 列車延誤未達門檻（各等級門檻不同）
  • 網路連線問題或 LINE 通知設定
  • 系統正在 5 分鐘檢查週期中
Q10：資料多久更新一次？

系統每 5 分鐘檢查一次所有活躍的到站提醒，確保即時性與準確性。這個頻率在準確性和 API 使用效率間取得最佳平衡。

Q11：支援哪些車站？

支援台鐵全線車站，包括：

  • 西部幹線：基隆～枋寮
  • 東部幹線：八堵～台東
  • 各支線車站
  • 總計 150+ 個主要車站
Q12：可以同時設定幾個提醒？

依會員等級有不同的額度限制：

  • 免費會員：1 個提醒
  • 標準會員：2 個提醒
  • 進階會員：5 個提醒
Q13：如何查看我的常用車次？

當您設定到站提醒時，系統會自動在選單頂部顯示您的常用車次。這些推薦基於您的歷史使用記錄，讓重複設定更加快速便利。

Q14：誤點多嚴重才會通知？

依會員等級有不同的通知門檻：

  • 免費會員：誤點超過 10 分鐘才通知
  • 標準會員：誤點超過 5 分鐘就通知
  • 進階會員：誤點超過 1 分鐘就通知
Q15：如何切換到新的卡片式介面？

新的卡片式介面已自動啟用，無需手動切換。當您使用功能時會自動顯示現代化的卡片設計，提供更好的視覺體驗和操作便利性。

Q16：資料來源是否準確可靠？

非常可靠！我們直接使用台鐵官方 API 作為唯一資料來源：

  • 官方來源：串接交通部台灣鐵路管理局官方資料 API
  • 即時查詢：每次檢查都直接查詢台鐵官方 API 取得最新資料
  • 準確一致：誤點資訊與台鐵官方發布的資料保持一致
  • 權威來源：避免第三方資料可能的不準確問題

您可以信賴我們提供的誤點資訊，因為這就是台鐵官方的第一手資料。

Q17：三卡片幫助系統在哪裡？

輸入「說明」即可看到重新設計的三卡片幫助介面，分別介紹基本功能、進階功能和會員方案。
